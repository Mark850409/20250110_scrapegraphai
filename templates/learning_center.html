爬蟲工具功能總覽

我們的爬蟲工具提供多種功能，幫助您輕鬆完成網路資料收集工作：

腳本生成

AI 智能生成 Python 爬蟲腳本，無需編程經驗。

智能爬蟲

自動分析網頁結構，即時爬取所需資料。

腳本生成功能說明

使用步驟：

  1. 輸入目標網站 URL
  2. 描述需要爬取的資料類型
  3. 選擇適合的提示詞模板
  4. 點擊生成按鈕
  5. 檢視並下載生成的腳本
使用技巧
  • 提供詳細的資料需求描述，可提高腳本準確性
  • 可以參考提示詞範本來優化您的需求描述
  • 生成後可以預覽腳本並進行必要的調整

智能爬蟲功能說明

使用步驟：

  1. 輸入目標網站 URL
  2. 選擇要爬取的資料類型
  3. 設定爬取參數（可選）
  4. 開始智能爬取
  5. 查看即時爬取結果
功能特點
  • 自動分析網頁結構，識別關鍵資訊
  • 支援動態加載內容的網頁
  • 可設定爬取頻率和範圍
  • 提供多種資料導出格式

執行腳本功能說明

使用步驟：

  1. 選擇要執行的爬蟲腳本
  2. 設定執行參數
  3. 啟動腳本執行
  4. 監控執行進度
  5. 查看執行結果和日誌
注意事項
  • 執行前請確認參數設置正確
  • 可以設定執行頻率和時間
  • 支援批次處理多個腳本
  • 提供執行日誌和錯誤提示

腳本生成紀錄功能說明

主要功能：

  • 查看歷史生成的腳本
  • 編輯和更新已有腳本
  • 複製和重用成功的腳本
  • 管理和刪除腳本記錄
管理建議
  • 定期整理不需要的腳本記錄
  • 為重要腳本添加標記和說明
  • 可以匯出腳本備份

智能客服工具功能說明

功能特點：

  • 智能回答使用者問題
  • 提供技術支援和指導
  • 自動學習和優化回答
  • 支援多種互動方式

自定義提示詞功能說明

使用步驟：

  1. 選擇或創建提示詞模板
  2. 自定義提示詞內容
  3. 測試提示詞效果
  4. 保存和管理提示詞
優化建議
  • 使用清晰具體的描述
  • 包含必要的格式要求
  • 添加示例和上下文
  • 定期更新和優化提示詞
